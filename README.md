# Personal Kanban Board

A small React 19, TypeScript, Vite, Tailwind CSS, and shadcn/ui learning app.

## What It Demonstrates

- Component composition with typed props
- Controlled forms and validation
- Local storage persistence
- `useMemo`, `useCallback`, `memo`, custom hooks, and `useReducer`
- Pure board state transformations separated from React rendering

## Run

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run build
npm run test
```
